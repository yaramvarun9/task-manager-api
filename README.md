Task Manager API (Spring Boot)

A simple and robust Task Management REST API built using Spring Boot.
This application supports full CRUD operations on tasks and follows a clean, layered architecture suitable for learning or portfolio projects.

Architecture

The project follows a standard three-layer architecture:

1. Controller Layer

Handles incoming HTTP requests and delegates processing to the service layer.

2. Service Layer

Contains business logic and validation.

3. Repository Layer

Handles data persistence using Spring Data JPA.
